## Crop Classification
This git repo includes the code for our study "**Deep Learning for Crop Classification Using Multi-Source Satellite Data**". Each part does the following : 

### AARN_Part1: 
Implementation of the base classification model "**MCTNet**" proposed by the article, this model gets an OA score of **0.977**.

### AARN_Part2: 
Evaluation of the model when adding enviromental covariates such as climate, soil, and topographic data. All data sources is included under the ressources section.

### AARN_Part 3 : 
This one is split into three models :
  - InceptionTime
  - IncepTAECovLF
  - HybridCropNet
